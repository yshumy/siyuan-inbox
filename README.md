# SiYuan Inbox Plugin

**Inbox** is a powerful collection tool for SiYuan Note. It allows you to quickly "dispatch" selected blocks, document titles, or text snippets into predefined target documents with custom separators and source tracking.

## âœ¨ Features

- **Multi-target Dispatch**: Choose different inboxes directly from the context menu.
- **Smart Hierarchical Filtering**: Automatically includes sub-content when a heading is selected and prevents duplicate blocks.
- **Text Snippet Collection**: Support for collecting specific text selections within a block.
- **Custom Boundaries**:
  - **Empty Heading**: Create clean visual separators (e.g., `# ---`).
  - **Serial Numbers**: Automatically calculate the next number in the target document (e.g., `0001`, `0002`).
  - **Timestamps**: Record the exact time of collection.
- **Automatic Source Tracking**: Adds a reference link to the original document at the end of the collected content.

## ğŸš€ Installation

1. Download the latest `package.zip` from the [Releases](https://github.com/manus/siyuan-inbox/releases) page.
2. In SiYuan Note: `Settings` -> `Appearance` -> `Open Appearance Directory`.
3. Navigate to the `plugins` folder and create a new folder named `inbox`.
4. Extract the contents of `package.zip` into the `inbox` folder.
5. Enable **Inbox** in `Settings` -> `Plugins`.
