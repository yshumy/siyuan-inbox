# SiYuan Inbox Plugin

**Inbox** 是一个为您量身定制的思源笔记高效收集插件。它可以将选中的块、文档标题或部分文字快速“投递”到预设的目标文档中，并支持自定义分割符和来源追踪。

## ✨ 功能特性

- **多目标投递**：在右键菜单中直接选择不同的收集箱（如：隨記、古音、正字法等）。
- **智能层级过滤**：选中标题时自动包含其下所有内容，并智能剔除重复块，确保导出内容完整且唯一。
- **文字片段收集**：支持在块内选中部分文字进行精准收集。
- **自定义分割边界**：
  - **空标题模式**：生成纯粹的标题分割线（如 `# ---`），视觉清爽。
  - **递增序号**：自动计算目标文档中的序号（如 `0001`, `0002`）。
  - **时间戳**：记录收集的具体时间。
- **来源自动追踪**：自动在收集内容末尾添加指向原文档的引用链接。

## 🚀 安装方法

1. 从 [Releases](https://github.com/manus/siyuan-inbox/releases) 页面下载最新的 `package.zip`。
2. 在思源笔记中：`设置` -> `外观` -> `打开外观目录`。
3. 进入 `plugins` 文件夹，新建一个名为 `inbox` 的文件夹。
4. 将 `package.zip` 中的内容解压到 `inbox` 文件夹内。
5. 在思源笔记的 `设置` -> `插件` 中启用 **Inbox**。

## 🛠️ 开发与配置

目前目标文档 ID 需在源码 `src/index.ts` 中配置。未来将支持通过插件设置界面进行可视化配置。

```bash
# 安装依赖
npm install

# 编译项目
npm run build
```
